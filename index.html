Hands Limited API Documentation

This API provides endpoints for user management, testimonials, and company updates.

Authentication

POST /auth/register

Register a new user account.

POST /auth/login

Authenticate user and receive JWT token.

Users

GET /users

Fetch all users (Admin only)

GET /users/:id

Fetch specific user details

PUT /users/:id

Update user information

DELETE /users/:id

Delete user account (Admin only)

Testimonials

GET /testimonials

Fetch all testimonials

GET /testimonials/:id

Fetch specific testimonial

POST /testimonials

Create new testimonial (Authenticated users)

DELETE /testimonials/:id

Delete testimonial (Admin only)

Updates

GET /updates

Fetch all company updates

POST /updates

Create new update (Admin only)

PUT /updates/:id

Edit existing update (Admin only)

DELETE /updates/:id

Delete update (Admin only)
