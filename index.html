Welcome to the API Documentation

This API provides a variety of endpoints to interact with our application. Below is a list of available endpoints and their descriptions.

Authentication

POST /auth/login

Authenticates a user and returns a JWT token.

Users

GET /users

Fetches a list of users.

POST /users

Creates a new user.

Categories

GET /categories

Fetches a list of categories.

POST /categories

Creates a new category.

Restaurants

GET /restaurants

Fetches a list of restaurants.

POST /restaurants

Creates a new restaurant.

Orders

GET /orders

Fetches a list of orders.

POST /orders

Creates a new order.

Addresses

GET /addresses

Fetches a list of addresses.

POST /addresses

Creates a new address.

Drivers

GET /drivers

Fetches a list of drivers.

POST /drivers

Creates a new driver.

Menu Items

GET /menu_items

Fetches a list of menu items.

POST /menu_items

Creates a new menu item.

Comments

GET /comments

Fetches a list of comments.

POST /comments

Creates a new comment.
